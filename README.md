# CAF SoftSol Company Name Normalizer (C)

## Problem
Employee data contains different formats of company name like:
- CAF softsol
- CAF solution
- CAF softSolution India Pvt Limited

## Solution
A C program that converts all formats into:
CAF SoftSol India Pvt Ltd.
